# 截面相关性 Alpha 策略调研报告

**调研日期**: 2026-06-25  
**背景**: 基于小资金约束（日频数据、≤1000只微盘股、有限计算资源）

---

## 1. 什么是截面相关性 Alpha

传统单因子选股：只看A股票自己的指标（市值、PB等）  
截面相关性选股：看A与B、C、D...的**关系**，从关系中发现定价偏差

**核心逻辑**：
- 如果A和B是"同类股"（同行业、同概念、同市值区间），它们的走势应该相关
- 当A突然偏离B的走势，可能是因为A被**过度反应**或**反应不足**
- 这种偏离 = alpha 机会

---

## 2. 四大截面相关性策略类型

### 2.1 配对交易（Pair Trading / Statistical Arbitrage）

| 项目 | 详情 |
|:--|:--|
| **原理** | 两只相关性高的股票，价差会均值回归 |
| **A股适用性** | ⭐⭐⭐ 中等 |
| **实施难度** | 中等（需要协整检验） |
| **资金要求** | 需要同时做多+做空，散户只能做"单腿" |
| **数据需求** | 日频即可 |
| **计算量** | 低 |
| **局限** | 散户无法融券做空，只能做"伪配对"：买低估的，卖高估的（需要已有持仓） |

**可落地变体——伪配对交易**：
```
1. 找到你持仓中相关性高的股票对
2. 当A相对B偏离2σ以上，卖出部分A，买入B
3. 等待价差回归
4. 调仓回原始配比
```
> 注意：这不是真正的市场中性，只是"持仓再平衡"

---

### 2.2 Lead-Lag 领先-滞后效应

| 项目 | 详情 |
|:--|:--|
| **原理** | 大盘股/行业龙头先动，小盘股滞后跟随 |
| **A股适用性** | ⭐⭐⭐⭐ 高（A股散户追涨龙头，机构调仓有顺序） |
| **典型场景** | 茅台涨 → 五粮液滞后涨；宁德时代涨 → 二线锂电滞后 |
| **微盘股适用** | ⭐⭐⭐ 中-高（微盘股常被作为"补涨"标的） |
| **数据需求** | 日频 |
| **计算量** | 低（交叉相关时滞分析） |

**具体做法**：
1. 选定"领先指标"：行业ETF、大盘指数、或行业龙头
2. 计算领先指标与每只微盘股的交叉相关系数（带时滞）
3. 当领先指标N日前大涨，而微盘股还未反应 → 买入
4. 持有直到微盘股"补涨"完成

**A股实证**：
- 清华PBCSF研究：A股存在显著的行业内部领先-滞后（t-stat > 3.0）
- 领先期：1-5个交易日
- 但2020年后随着量化普及，lead-lag alpha在衰减

### 2.3 Peer Momentum（同伴动量）

| 项目 | 详情 |
|:--|:--|
| **原理** | 不是看股票自己的动量，而是看"同伴"（peer）的动量 |
| **同伴定义** | 同行业、同市值区间、同地区、或PCA提取的共同因子暴露 |
| **A股适用性** | ⭐⭐⭐⭐ 高 |
| **微盘股适用** | ⭐⭐⭐⭐ 高（微盘股易被同伴带动） |
| **数据需求** | 日频 + 行业分类 |
| **计算量** | 中（需要计算peer portfolio return） |

**具体做法**：
1. 对每只股票，定义其"peer group"（同行业市值最接近的10只）
2. 计算peer group过去20日的平均收益
3. 因子 = peer_momentum（peer的收益）
4. 选股：选peer momentum高但自身还没涨的股票（补涨逻辑）

**文献支持**：
- "Peer momentum" (Moskowitz & Grinblatt, 1999, JF): 行业动量本身就是alpha
- A股验证：行业动量效应显著，但1月内最强，3月后衰减

---

### 2.4 图神经网络（GNN）截面关系建模

| 项目 | 详情 |
|:--|:--|
| **原理** | 把股票看作图的节点，关系（同行业、同概念、供应链等）作为边，用GNN学习截面定价偏差 |
| **A股适用性** | ⭐⭐ 低-中（概念复杂，但数据获取难） |
| **微盘股适用** | ⭐⭐ 低（微盘股信息更少，图关系更弱） |
| **数据需求** | 日频 + 股票关系图谱（行业、概念、供应链、股东重叠等） |
| **计算量** | **极高**（需要GPU、大规模训练） |
| **可落地性** | 对小资金：❌ 不现实 |

**为什么GNN对小资金不现实**：
- 需要构建高质量的股票关系图（数据获取成本大）
- 需要GPU训练（一张3090要几千块）
- 需要大量数据才能训练出稳定模型（过拟合风险极高）
- 模型解释性差，实盘出问题难以调试

## 3. 小资金可落地的截面相关性策略

### 推荐方案：Peer Momentum + Lead-Lag 组合

**为什么选这个**：
- 数据需求低（只需日频 + 行业分类）
- 计算量可控（Python + pandas 即可）
- 逻辑可解释（同行补涨、龙头带动）
- 不需要做空（单腿做多即可）

**实施步骤**：

```python
# 1. 构建行业-市值分组
for 每个行业:
    按市值分为 5 组（大/中/小/微/极小）
    
# 2. 计算 Peer Momentum
for 每只股票:
    peer_group = 同行业同市值组的最近10只（排除自己）
    peer_mom = peer_group过去20日平均收益
    
# 3. 计算 Lead-Lag
for 每只股票:
    leader = 同行业市值最大的前3只（龙头）
    leader_mom = leader过去5日平均收益
    lag = 交叉相关系数峰值对应的时滞天数
    
# 4. 合成因子
composite_signal = 0.5 * peer_mom + 0.3 * leader_mom_5d - 0.2 * own_mom20d

# 5. 选股
选 composite_signal 最高但 own_mom20d 最低的股票（补涨逻辑）
```

**预期效果**：
- 在牛市/震荡市中：peer_momentum 可以捕捉到轮动节奏
- 在熊市中：可能失效（因为所有股票都在跌，没有"补涨"）
- 年化增益：+3-5%（相比纯市值因子）

## 5. 实施建议

### 阶段1：验证Peer Momentum因子（1周）
1. 在 `backtest_mvp` 中新增 `peer_momentum` 因子计算
2. 测试单因子IC（信息系数）
3. 如果IC > 0.03，进入阶段2

### 阶段2：组合到现有策略（2周）
1. 将 `peer_momentum` 作为复合因子之一加入S5/SB
2. 权重：mcap(0.3) + pb(0.3) + peer_mom(0.2) + reversal(0.2)
3. Walk-forward验证：看是否提升OOS夏普

### 阶段3：Lead-Lag增强（2周）
1. 增加行业龙头动量作为先导信号
2. 仅当龙头动量>0时，才进入该行业的微盘股
3. 这相当于一个"行业择时"过滤器

---

## 6. 关键风险

1. **Peer Momentum 在市场极端分化时失效**：比如2021年新能源 vs 传统能源，同行不同命
2. **Lead-Lag 窗口不稳定**：领先期从1天到5天不等，难以固定
3. **计算 peer group 需要实时数据**：如果行业分类变化（如公司转型），需要更新
4. **小盘股流动性差**：peer momentum信号发出后，可能买不到足够的量

---

## 7. 总结

| 结论 | 说明 |
|:--|:--|
| **最推荐** | Peer Momentum + Lead-Lag 组合 |
| **数据需求** | 日频 + 行业分类（已有） |
| **计算量** | 中等（pandas groupby + rolling） |
| **预期增益** | +3-5% 年化（相比纯市值） |
| **优先实施** | 先在现有策略中增加 peer_momentum 因子，验证IC |
| **不推荐的** | GNN（计算量过大、数据需求过高、过拟合风险） |
